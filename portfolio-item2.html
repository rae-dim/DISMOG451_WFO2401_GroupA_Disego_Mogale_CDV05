Kanban Agile Board

A board for organizing tasks

Tech stack used to achieve this project includes HTML, CSS and JavaScript

This agile board is meant to assist users in planning out tasks. The app allows one to add different sections of tasks, mark tasks either as done, doing or to-do. As an added benefit, the app also features a toggle button that allows users to switch between light and dark mode.
